# binary-learning-toy
A toy to help children and adults learn binary

### Pin connections
```
1	Switch128
0	Switch64
				
2	Switch32	A3	SegmentF
3	Switch16	A2	SegmentE
4	Switch8		A1	SegmentD
5	Switch4		A0	SegmentC
6	Switch2		13	SegmentB
7	Switch1		12	SegmentA
8	Digit1		11	Digit4
9	Digit2		10	Digit3

A4	SegmentG
A5	SegmentDP
A6	Switch Signed
A7	Switch Dec/Hex

binary switches		8
lower switches		2
digit pins		4
segment pins		8

total			22
```
